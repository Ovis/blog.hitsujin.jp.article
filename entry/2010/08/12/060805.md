バージョン管理に利用されるSubversionが作る.svnディレクトリを隠す方法。

``` conf
RewriteEngine On  
RewriteRule ^(.*/)*.svn/ / [F,L]
```
httpd.confをいじれる環境の場合は

``` conf
<Directory ~ ".svn">
	Order allow,deny 
	Deny from all  
</Directory>
```
